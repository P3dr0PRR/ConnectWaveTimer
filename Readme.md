# ConnectWaveTimer

O **ConnectWaveTimer** é uma aplicação de cronômetro baseada na metodologia **Pomodoro**, desenvolvida a partir do projeto **Ignite Timer**, da trilha de **ReactJS** do programa Ignite da Rocketseat.

Este projeto foi adaptado e expandido por mim com o objetivo de reforçar conceitos fundamentais e avançados de React, especialmente **TypeScript**, **Rotas (React Router)** e **estilização com Tailwind CSS**, além de consolidar boas práticas de organização e arquitetura front-end.

> O projeto ainda está em desenvolvimento.

---

## Objetivo do Projeto

O principal objetivo do **ConnectWaveTimer** é aprofundar o aprendizado em:

- Tipagem forte com TypeScript
- Navegação e organização de páginas com React Router
- Gerenciamento de estado em aplicações React
- Estruturação de projetos escaláveis
- Estilização moderna e produtiva com Tailwind CSS

---

## Funcionalidades

- **Gestão de Ciclos**
  - Criação de tarefas personalizadas
  - Definição de tempo de foco para cada ciclo

- **Contagem Regressiva**
  - Timer visual para controle do tempo
  - Indicação clara do ciclo ativo

- **Histórico de Atividades**
  - Registro de tarefas realizadas
  - Status das tarefas: concluídas, interrompidas ou em andamento

- **Persistência de Dados**
  - Armazenamento local para manter o histórico mesmo após recarregar a aplicação

---

## Tecnologias e Conceitos Utilizados

### React e TypeScript

- Componentização
- Tipagem estática para maior segurança e legibilidade
- Melhor previsibilidade do código

### Gerenciamento de Estado

- useState
- useEffect
- useReducer para estados mais complexos

### Rotas

- Navegação entre páginas utilizando React Router
- Separação clara de responsabilidades entre telas

### Estilização

- Tailwind CSS para criação de layouts rápidos, responsivos e consistentes
- Foco em produtividade e manutenção simples do CSS

### Formulários e Validação

- React Hook Form
- Zod para validação de dados e tipagem segura
